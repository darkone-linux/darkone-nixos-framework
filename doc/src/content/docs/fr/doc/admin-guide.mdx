## Démarrage



### Installation

On va installer dans l'ordre :

1. une machine (celle de l'administrateur),
2. une passerelle et d'autres machines dans la même zone,
3. le réseau tailnet (serveur de coordination),
4. d'autres zones liées à notre configuration.

:::note[Une configuration unique]
L'ensemble de ces noeuds seront décrits dans une unique configuration NixOS que l'on va administrer sur la machine de l'administrateur.
De cette manière, on va pouvoir mettre à jour et faire évoluer tout le réseau et ses machines de manière cohérente et centralisée.
:::
